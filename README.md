## Installation

### Requirements
```
Ruby 2.5.0
Rails 5.1.4
Postgres 9.5
```
Start by forking and cloning your fork
```
git clone https://github.com/YOUR-USERNAME/UWranks.git
cd UWranks.git
```
Install dependencies
```
bundle install
```
Setup Local Database
```
rake db:create
rake db:migrate
```
Run the Server
```
rails server
```
Navigate to https://localhost:3000
